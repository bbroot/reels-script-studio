# 六大类型模板（Templates · v3.5 数据增强）

> 每类模板的镜头侧重、台词范式与**爆款因子绑定**。时间轴默认 15 秒（0–2 钩子 / 2–5 高光 / 5–9 演示 / 9–12 对比 / 12–15 CTA），时长可整体缩放。
> 爆款因子定义见 `viral-playbook.md`：钩子原型（第三节）、四大共鸣情绪（第四节）、转发钩子（第八节）、平台数据（第六节）。

## A. 产品种草型（美妆/3C/家电）
- **侧重**：卖点前置 + 质感特写 + 价格锚点。
- **镜头**：1 【钩子原型：痛点暴击/数字冲击】→ 2 微距探针产品质地 → 3 机械臂 360° 亮相 → 4 使用前后对比 → 5 CTA + 【转发钩子：实用囤货】。
- **台词范式**："还在为 XX 烦恼？这瓶 XX，7 天逆转。" / "现在入手，到手价 XX——先收藏，手慢无。"
- **爆款绑定**：钩子原型(痛点/数字) + 情绪锚点(欣赏美好) + 转发钩子(实用囤货)。

## B. 知识科普型（教育/职场/健康）
- **侧重**：反常识开场 + 信息密度 + 权威收尾 + **可收藏价值**。
- **镜头**：1 【钩子原型：反常识】→ 2 数据/图表动画 → 3 原理拆解 → 4 行动建议(给「15秒解决1痛点」点) → 5 关注 + 【转发钩子：身份认同/实用囤货】。
- **台词范式**："你以为 XX 是对的？其实 90% 的人都错了。" / "记住这 3 点，下次不踩坑——先收藏。"
- **爆款绑定**：钩子原型(反常识) + 五大公式(实用性干货 长尾+120%) + 转发钩子(实用囤货)。

## C. 剧情叙事型（品牌故事/情感微电影）
- **侧重**：情绪铺垫 + 人物弧光 + 品牌自然植入 + **爽感逆袭**。
- **镜头**：1 【钩子原型：情绪直给/悬念冲突】→ 2 回忆/闪回 → 3 转折(反转/打脸) → 4 品牌价值落点 → 5 CTA + 【转发钩子：情绪共鸣】。
- **台词范式**："他曾以为……直到遇见 XX。" / "陪你熬过最难那年的，是它。"
- **爆款绑定**：钩子原型(情绪/悬念) + 情绪锚点(忧郁伤感/青春回忆) + 转发钩子(情绪共鸣) + 完播目标≥70%。

## D. Vlog 生活型（旅行/日常/探店）
- **侧重**：第一视角沉浸 + **真实感溢价**(素颜生活流) + 轻松节奏 + 本地标签。
- **镜头**：1 环境空镜【钩子原型：极致对比/情绪直给】→ 2 第一视角行走 → 3 体验高潮 → 4 不经意种草 → 5 关注 + 【转发钩子：身份认同】。
- **台词范式**："今天带你们来一个宝藏地方。" / "上海打工人的夜，大概就这样。"
- **爆款绑定**：真实感溢价(粗糙真实) + 本地化标签(#城市, CTR+55%) + 情绪锚点(青春回忆/欣赏美好)。

## E. 测评对比型（好物推荐/避坑）
- **侧重**：客观对比 + 分屏 + 结论清晰 + 社交货币。
- **镜头**：1 【钩子原型：身份反差/数字冲击】→ 2 维度一对比 → 3 维度二对比 → 4 分屏总结 → 5 购买建议 + 【转发钩子：社交货币/争议留白】。
- **台词范式**："XX 和 XX 到底差在哪？实测告诉你。" / "预算有限选 A，追求极致选 B——你站哪边？评论区吵。"
- **爆款绑定**：钩子原型(身份反差/数字冲击 互动9.6%) + 分屏对比 + 转发钩子(争议留白)。

## F. 热点评论型（时事/观点/行业分析）
- **侧重**：时效钩子 + 观点鲜明 + 引发讨论 + 热点 BGM。
- **镜头**：1 【钩子原型：悬念冲突】+ #挑战名 → 2 你的立场 → 3 论据支撑 → 4 反转/升华 → 5 评论区互动 + 【转发钩子：争议留白/社交货币】。
- **台词范式**："最近 XX 刷屏了，但很少有人提到这一点。" / "你怎么看？转发让你的朋友也想想。"
- **爆款绑定**：热点BGM/挑战(曝光4–7×) + 转发钩子(争议留白) + 情绪锚点(家国情怀/青春回忆)。

## ➕ AI 创意型（第七赛道，2026 风口）
- 适用：AI 漫剧 / AI 动画 / AI 工具教程（素材显示「AI融入创作」「AI赋能」成新风口）。
- **侧重**：技术好奇+焦虑感 + 未来向往 + 强视觉。
- **镜头**：1 【钩子原型：反常识/数字冲击】→ 2 AI 生成过程特写 → 3 效果炸裂展示 → 4 对比(有无AI) → 5 引导关注/试用 + 转发钩子(社交货币)。
- **爆款绑定**：五大公式(紧跟热点) + 情绪锚点(欣赏美好) + 转发钩子(显得你有品味)。
- **提示**：可在 1️⃣ 视频类型里归入 B(科普)或 F(热点)，第 6 维勾选 A/D/F 强化。
