# 六大类型模板（Templates）

> 每类模板的镜头侧重与台词范式。时间轴默认 15 秒（0–2 钩子 / 2–5 高光 / 5–9 演示 / 9–12 对比 / 12–15 CTA），时长可整体缩放。

## A. 产品种草型（美妆/3C/家电）
- **侧重**：卖点前置 + 质感特写 + 价格锚点。
- **镜头**：1 痛点钩子 → 2 微距探针产品质地 → 3 机械臂 360° 亮相 → 4 使用前后对比 → 5 CTA 下单。
- **台词范式**："还在为 XX 烦恼？这瓶 XX，7 天逆转。" / "现在入手，到手价 XX。"

## B. 知识科普型（教育/职场/健康）
- **侧重**：反常识开场 + 信息密度 + 权威收尾。
- **镜头**：1 反常识提问 → 2 数据/图表动画 → 3 原理拆解 → 4 行动建议 → 5 关注收藏。
- **台词范式**："你以为 XX 是对的？其实 90% 的人都错了。" / "记住这 3 点，下次不踩坑。"

## C. 剧情叙事型（品牌故事/情感微电影）
- **侧重**：情绪铺垫 + 人物弧光 + 品牌自然植入。
- **镜头**：1 情绪特写 → 2 回忆/闪回 → 3 转折 → 4 品牌价值落点 → 5 情感 CTA。
- **台词范式**："他曾以为……直到遇见 XX。" / "陪伴，是最长情的告白。"

## D. Vlog 生活型（旅行/日常/探店）
- **侧重**：第一视角沉浸 + 真实感 + 轻松节奏。
- **镜头**：1 环境空镜钩子 → 2 第一视角行走 → 3 体验高潮 → 4 不经意种草 → 5 关注。
- **台词范式**："今天带你们来一个宝藏地方。" / "真的被惊艳到了。"

## E. 测评对比型（好物推荐/避坑）
- **侧重**：客观对比 + 分屏 + 结论清晰。
- **镜头**：1 选题钩子 → 2 维度一对比 → 3 维度二对比 → 4 分屏总结 → 5 购买建议。
- **台词范式**："XX 和 XX 到底差在哪？实测告诉你。" / "预算有限选 A，追求极致选 B。"
- **默认加**：分屏对比设计（进化经验：用户常手动改第 4 镜为分屏）。

## F. 热点评论型（时事/观点/行业分析）
- **侧重**：时效钩子 + 观点鲜明 + 引发讨论。
- **镜头**：1 热点事件钩子 → 2 你的立场 → 3 论据支撑 → 4 反转/升华 → 5 评论区互动。
- **台词范式**："最近 XX 刷屏了，但很少有人提到这一点。" / "你怎么看？评论区聊聊。"
