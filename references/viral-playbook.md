# 爆款因子手册（数据驱动 / Viral Playbook）

> 来源：蝉妈妈 2025Q4 白皮书、新榜 2025 复盘、抖音运营机制（2025–2026）。
> 本手册是「显著提升爆款潜力」的方法论底座，被 SKILL.md / templates.md / platform-adapters.md 引用。
> 所有数字均为行业实测，用于自检与参数化，而非承诺。

## 一、黄金 3 秒法则（头号生死线）

- 用户平均停留时长仅 **1.7 秒**；**超过 60% 观众在前 3 秒划走**。
- 前 3 秒完播率 < **32%** 的视频，进入推荐池概率 < **8%**。
- 注意力窗口从 2024 年的 8s 降至 2025 年的 **4.5s** → 开头必须「钉住眼球」，而非画面精美。
- **结论**：每支脚本第 1 镜强制绑定一种「钩子原型」（见第三节），并在质量自检中给出 3 秒留存目标。

## 二、五大爆款公式（蝉妈妈验证）

1. **强情绪驱动**：情感+搞笑类完播率 **78%**；前 3 秒必须推高能情节。
2. **实用性干货**：长尾词搜索量同比 +**120%**（手机技巧/省钱妙招）；「15 秒解决 1 个痛点」已验证。
3. **反差感人设**：身份与行为巨大反差；「逆袭」「土豆」标签互动率 **9.6%**（普通视频 2.3×）。
4. **紧跟热点 BGM/挑战**：使用当周热门 BGM，曝光量比普通视频高 **4–7 倍**。
5. **本地化+生活化记录**：带城市标签视频在本地推荐页点击率 +**55%**。

## 五、七大共性规律（必过清单）

1. **情绪价值驱动**：缺情绪 = 缺传播力。每支脚本至少绑定 1 个情绪锚点。
2. **前 3 秒定生死**：见第一节，强制钩子原型。
3. **真实感溢价**：精致人设失灵，「粗糙真实」成稀缺品；素颜 Vlog / 生活流狂揽流量 → 光影风格提供「真实粗糙」选项。
4. **社交货币属性**：内容必须让用户「愿意转发」→ CTA 强制含「转发钩子」（搞笑/观点/攻略最易传播）。
5. **实用即正义**：收藏率成算法新权重 → 实用类必须有「可收藏」价值点。
6. **垂直深挖胜于广泛**：窄众赛道更易打爆（锻刀/速降/鉴蛇）→ 支持「子赛道」标注。
7. **AI 赋能创作**：AI 漫剧/动画/工具教程成风口 → 模板内置 AI 创意用法。

## 七、完播/互动目标值（质量自检量化基准）

- 情绪/搞笑类：完播率目标 ≥ **70%**（实测均值 78%）
- 实用/知识类：完播率目标 ≥ **55%**，必须有收藏钩子
- 搞笑/观点类：互动率（赞+评+转）目标 ≥ **9.6%**
- 前 3 秒：钩子原型命中 + 完播目标 ≥ **45%**（普通视频基准 32%）
- 连续 3 条完播 <35% → 流量池降级（新号前 5 条无爆款 60% 变僵尸）→ 强调「稳定输出 + 首条即爆」

## 八、转发钩子库（CTA 必备）

- 争议留白："你觉得呢？评论区吵起来。"
- 身份认同："同为打工人，这条必须转给同事。"
- 实用囤货："先收藏，用的时候找不到就亏了。"
- 情绪共鸣："转给那个总让你熬夜的人。"
- 社交货币："转发装 X / 显得你有品味"型内容。

## 九、热点 BGM / 挑战策略

- 生成时若用户未指定，标注「建议当周热门 BGM 类型」（如卡点/治愈/悬疑），并提示曝光 4–7×。
- 支持「挑战贴」结构：开头 #挑战名，降低发现成本。
- 注意时效：BGM 热度窗口约 7–15 天，慢热期内容可借势长尾。

## 十、本地化标签策略

- 生活/探店/美食类强制带城市标签（如 #北京美食），本地推荐页 CTR +55%。
- 情绪/Vlog 可带「城市+状态」标签增强真实感（#上海打工人的夜）。
